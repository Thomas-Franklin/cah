# Getting started

`npm i -g`

Now we can perform `cah`:

# Help

```
cah --help
cah [command]

Commands:
  cah random [type]  produce random card
  cah deal           deal a random combination

Options:
  --help         Show help                                             [boolean]
  --version      Show version number                                   [boolean]
  --verbose, -v  Run with verbose logging                              [boolean]
```
